## 📱 **행사 당일 운영 가이드**

### 준비사항 체크리스트
- [ ] WiFi 환경 및 인터넷 연결 확인
- [ ] 관리자 디바이스 배터리 충분한지 확인

### 예상 문제 및 해결방안
1. **QR 스캔 안됨**: 직접 URL 입력 안내
2. **인터넷 불안정**: 오프라인 모드로 전환
3. **디바이스 문제**: 다른 기기에 데이터 복사
4. **대량 체크인**: 여러 디바이스 동시 사용
