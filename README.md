# 오산고 71회 동기 송년회 초대장 시스템

프리미엄 초대장과 QR 체크인 시스템을 갖춘 동기회 관리 시스템입니다.

## ✨ 주요 기능

- 🎨 **프리미엄 초대장**: 우아한 디자인의 초대장 with QR 코드
- 📱 **QR 체크인 시스템**: 모바일 친화적인 체크인 프로세스
- 👥 **자동 팀배정**: 체크인 시 자동으로 팀 배정
- 📊 **관리자 대시보드**: 참석자 현황 실시간 확인 및 관리

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS (CDN)
- **Data**: localStorage (클라이언트 사이드)
- **QR Code**: QR Server API
- **Deployment**: Vercel

## 📱 페이지 구성

1. **초대장 페이지** (`/invitation`)
   - 행사 정보 및 QR 코드 표시
   - 모바일 최적화된 반응형 디자인

2. **체크인 페이지** (`/checkin`)
   - QR 스캔 후 접근하는 체크인 폼
   - 자동 팀 배정 시스템

3. **관리자 대시보드** (`/admin`)
   - 참석자 목록 및 현황
   - 정렬 기능 (이름/체크인/시간순)
   - CSV 내보내기 및 데이터 관리

## 🎉 행사 당일 운영 가이드

### 📱 **권장 운영 방식**

#### **단일 관리자 방식** (가장 간단)
1. **메인 관리자**: 하나의 디바이스(PC/태블릿)에서 전체 관리
2. **참석자 체크인**: 모든 디바이스에서 QR 스캔하여 체크인
3. **실시간 확인**: 관리자 디바이스에서 실시간 현황 모니터링

#### **다중 관리자 방식** (데이터 공유)
1. **데이터 내보내기**: 메인 관리자가 "📱 데이터 공유" 버튼 클릭
2. **클립보드 복사**: 자동으로 전체 데이터가 클립보드에 복사됨  
3. **다른 디바이스**: "📲 데이터 가져오기"로 붙여넣어서 동기화

### ⏰ **행사 진행 순서**
```
행사 시작 2시간 전
├── 관리자가 참석자 명단 사전 등록
├── QR코드 테스트 (모바일에서 스캔 확인)
└── 체크인 시스템 최종 점검

행사 당일 접수 시간
├── 참석자들이 QR코드로 체크인
├── 관리자가 실시간 현황 모니터링
└── 필요시 수동으로 추가 체크인

행사 종료 후
├── CSV 다운로드로 최종 명단 백업
├── 참석 통계 확인
└── 데이터 보관
```
