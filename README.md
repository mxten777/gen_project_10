# 오산고 71회 동기 송년회 초대장 시스템

프리미엄 초대장과 QR 체크인 시스템을 갖춘 동기회 관리 시스템입니다.

## ✨ 주요 기능

- 🎨 **프리미엄 초대장**: 우아한 디자인의 초대장 with QR 코드
- 📱 **QR 체크인 시스템**: 모바일 친화적인 체크인 프로세스
- 👥 **자동 팀배정**: 체크인 시 자동으로 팀 배정
- 📊 **관리자 대시보드**: 참석자 현황 실시간 확인 및 관리

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS (CDN)
- **Data**: localStorage (클라이언트 사이드)
- **QR Code**: QR Server API
- **Deployment**: Vercel

## 📱 페이지 구성

1. **초대장 페이지** (`/invitation`)
   - 행사 정보 및 QR 코드 표시
   - 모바일 최적화된 반응형 디자인

2. **체크인 페이지** (`/checkin`)
   - QR 스캔 후 접근하는 체크인 폼
   - 자동 팀 배정 시스템

3. **관리자 대시보드** (`/admin`)
   - 참석자 목록 및 현황
   - 정렬 기능 (이름/체크인/시간순)
   - CSV 내보내기 및 데이터 관리

## 🚀 배포 및 사용법

### Vercel 배포
```bash
npm install -g vercel
vercel --prod
```

### 로컬 개발
```bash
# Python HTTP Server (권장)
python -m http.server 8000 --bind 0.0.0.0

# 접속: http://localhost:8000
```

## 📱 모바일 지원

- 반응형 디자인으로 모든 디바이스 지원
- QR 코드 스캔 최적화
- 터치 인터페이스 친화적

## 🎯 주요 특징

- **환경 감지**: 로컬/배포 환경에 따른 QR 코드 URL 자동 생성
- **데이터 지속성**: localStorage를 활용한 오프라인 데이터 보관
- **실시간 업데이트**: 페이지 새로고침 없이 실시간 상태 반영
- **접근성**: ARIA 라벨 및 키보드 네비게이션 지원
